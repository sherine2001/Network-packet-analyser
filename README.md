# Network Packet Analyzer

A real-time network packet analyzer built with Python and Scapy.

## Features
- Captures live TCP/UDP/IP packets
- Displays source and destination IPs in real time
- Shows live statistics (TCP/UDP/Other count)
- Save captured packets to JSON log file
- Clean dark-themed UI

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run: python main.py
3. Click "Start Capture" (run as administrator/sudo)

## Technologies Used
- Python
- Scapy
- Tkinter
- Threading
